# SoftTech Analytics — Projeto de Estatística


## Recomendações

Recomenda-se criar um ambiente virtual para instalação das seguintes bibliotecas:

- Pandas
- OpenPyXL
- SciPy
- Pytest
- Matlotib
- Streamlit

### Passo a passo:

#### Passo 1 - Utilize o comando *cd* para mudar para o diretório em que deseja criar o ambiente virtual:
```
cd .../"SoftTech Analytics"
```

#### Passo 2 - Crie o ambiente virtual:
```bash
python -m venv .venv
```

#### Passo 3 - Ative o ambiente virtual (Windows):
```bash
.venv\Scripts\Activate.ps1

ou

.\.venv\Scripts\Activate.ps1
```

#### Passo 4 - Instale as dependências:
```
pip install -r requirements.txt
```
